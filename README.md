## Example for H1 header
```
<link href="https://fonts.cognicue.in/cognicue.css" rel="stylesheet" type="text/css" />

<style type="text/css">
h1 {
	font-family: CogniCue, sans-serif;
	font-weight: 700;
	font-style: normal;
	font-stretch: condensed;
}
</style>
```
